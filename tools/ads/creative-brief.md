# 🎨 Creative Brief — CertZen Social Media Ads
> Para diseñador o para crear en Canva | Todos los formatos de ads

---

## Identidad Visual de Referencia

Usa los estilos de la app (certzen.app) como guía:
- **Colores primarios:** ver Tailwind config del proyecto (`tailwind.config.js`) — tonos oscuros + accent brand-500
- **Tipografía:** moderna, sans-serif (Inter o similar)
- **Estilo:** glassmorphism sutil, contrastes altos, nada recargado
- **Tono visual:** profesional pero dinámico — NO corporativo aburrido

**Herramienta recomendada:** [Canva Pro](https://canva.com) o Figma  
**Plantillas útiles en Canva:** buscar "SaaS App Ad", "Tech Education", "Minimal Dark UI"

## CREATIVO 1 — Dashboard Screenshot Ad (IT)
> Objetivo: mostrar el producto real. Mejor conversión para audiences cálidas.

### Versión Feed Cuadrado (1080x1080)
```
LAYOUT:
- Fondo: dark (como la app) o gradiente oscuro azul/verde
- Centro: Screenshot real del dashboard de CertZen (resultados por dominio)
  → Mostrar la gráfica de radar o barras por dominio con scores reales
  → El screenshot debe verse limpio, no truncado
- Headline superpuesto (parte inferior): 
  "Sabe exactamente en qué dominio eres débil."
- Logo CertZen: esquina superior derecha, pequeño
- Sin precio en el creativo

TEXTO EN IMAGEN: máximo 20% del área total
FORMATO ARCHIVO: JPG o PNG, < 8MB
```

### Versión Story (1080x1920)
```
LAYOUT:
- Fondo: gradiente oscuro vertical
- Parte superior 40%: hook de texto
  "¿Listo para tu certificación IT?"
- Parte media 40%: screenshot del simulador en móvil (mockup de teléfono)
- Parte inferior 20%: CTA + URL
  "Empieza gratis → certzen.app"
- Swipe-up zone: dejar espacio libre en la parte inferior (no poner elementos importantes ahí)
```

## CREATIVO 6 — Video 15s (Reel + Story + Feed)
> Mayor engagement. Requiere herramienta de screen recording.

### Guión (15 segundos)
```
[0–3s] Texto animado: "¿Cuántos candidatos fallan su certificación IT?"
[3–5s] Número grande: "70%"  (efecto impacto)
[5–10s] Screen recording: usuario hace un examen en CertZen → ve resultados por dominio
[10–13s] Texto: "Ahora sabes exactamente dónde mejorar."
[13–15s] Logo + CTA: "certzen.app — Empieza gratis"

VOZ EN OFF (opcional): "El 70% falla porque no sabían dónde estaban débiles. CertZen lo cambia."
MÚSICA: upbeat pero no agresiva (usar música libre de derechos de Pixabay/YouTube Audio Library)
```

### Herramientas para hacer el video
- **Screen recording:** Loom (gratis), OBS, o la grabación nativa de tu sistema
- **Edición:** CapCut (móvil, gratis), DaVinci Resolve (desktop, gratis) o Canva Video
- **Formato de exportación:** MP4, H.264, < 500MB, ratio 9:16 (stories) o 1:1 (feed)

## Checklist antes de subir creativos

### Meta Ads
- [ ] Imagen cuadrada 1080x1080 ✓
- [ ] Texto en imagen < 20% del área (verificar con Meta Text Overlay tool)
- [ ] No incluir precios en la imagen (puede activar revisión manual)
- [ ] Resolución mínima: 1080x1080 px
- [ ] Archivo < 30MB
- [ ] No hay CTAs falsos (botones que no son botones reales)

### LinkedIn Ads
- [ ] Imagen horizontal 1200x627 ✓
- [ ] Texto en imagen mínimo — LinkedIn prefiere imágenes limpias
- [ ] Logo visible y en alta resolución
- [ ] Colores de marca consistentes
- [ ] No hay texto pixelado o borroso

### Videos
- [ ] Primeros 3 segundos tienen el hook (lo más importante)
- [ ] Funciona sin audio (añadir subtítulos)
- [ ] Ratio correcto: 9:16 para stories, 1:1 para feed, 16:9 para YouTube/LinkedIn
- [ ] Longitud: ≤ 15s para paid, ≤ 60s para orgánico
